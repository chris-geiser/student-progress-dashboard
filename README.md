# Student Progress Dashboard

Prototype dashboard for school administrators to track K-2 student reading progress across a district. Single self-contained HTML file, procedurally generated demo data, no build step.

Open `district-dashboard.html` in a browser to see the current state.

The full spec lives in [DASHBOARD-SPEC.md](DASHBOARD-SPEC.md). Read that before making changes.

## Files

- `district-dashboard.html`, the current prototype and canonical reference
- `DASHBOARD-SPEC.md`, design, data model, components, and decisions
- `school-dashboard.html`, an earlier single-school iteration. Out of sync with the district dashboard. Do not use as a reference.
